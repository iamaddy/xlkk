Module: storage

storage

Storage Lite provides a lightweight YUI 3 API for persistent cross-browser local key/value storage similar to the HTML5 localStorage API. Supported browsers include IE6+, Firefox 2+, Safari 3.1+, Chrome 4+, and Opera 10.5+. No browser plugins are required (not even Flash). 从YUI Storage修改而来
